This course covers development of web systems with a focus on client-side development. Topics include use of XHTML, the Document Object Model (DOM), Cascading Style Sheets (CSS), use of Javascript for providing client side interactivity, ADA accessibility, Javascript Object Notation (JSON), and design for mobile platforms. A web site design project is included.

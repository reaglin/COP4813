Drawing and Graphics with the HTML5 Canvas

This module introduces learners to the HTML5 <canvas> element and the rich 2D drawing API it exposes. Topics span foundational shape and path rendering, color and style management, text and image integration, and animation techniques — equipping students to build dynamic, scriptable graphics directly in the browser.
